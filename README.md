# signum-video

## Project setup
```
npm install
```

### Run prod mode
```
npm run prod
```

### Run auth server
```
npm run auth-server
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
